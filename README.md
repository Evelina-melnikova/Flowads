# FlowAds — Лендинг

Сайт агентства по продвижению в Яндекс.Картах.
https://evelina-melnikova.github.io/Flowads/
## Структура проекта

```
flowads/
├── index.html          — главная страница
├── css/
│   └── styles.css      — все стили (CSS переменные, адаптив, анимации)
├── js/
│   └── main.js         — скрипты (маска телефона, форма, scroll reveal, мобильное меню)
├── fonts/
│   └── README.md       — инструкция по подключению шрифта Inter
├── images/
│   ├── favicon.svg     — иконка сайта
│   ├── logo.svg        — логотип FlowAds
│   └── og-image.svg    — превью для соцсетей (OG Image)
└── README.md
```

## Технологии

- Чистый HTML5 + CSS3 + Vanilla JS (без фреймворков)
- CSS Custom Properties (переменные для цветов — легко менять тему)
- Адаптив: Desktop 1280px / Laptop 1100px / Tablet 860px / Mobile 520px
- Шрифт Inter подключён через Google Fonts CDN

## Цветовая палитра (CSS переменные)

```css
--bg:       #0D0D0F   /* фон страницы */
--surface:  #17171C   /* поверхность карточек */
--surface2: #1E1E24   /* альтернативная поверхность */
--purple:   #8B5CF6   /* основной акцент */
--blue:     #3B82F6   /* второй акцент */
--green:    #22C55E   /* позитивные метрики */
--text:     #FFFFFF   /* основной текст */
--text2:    #A1A1AA   /* второстепенный текст */
```
