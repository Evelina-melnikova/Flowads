# FlowAds — Лендинг

Сайт агентства по продвижению в Яндекс.Картах.

## Структура проекта

```
flowads/
├── index.html          — главная страница
├── css/
│   └── styles.css      — все стили (CSS переменные, адаптив, анимации)
├── js/
│   └── main.js         — скрипты (маска телефона, форма, scroll reveal, мобильное меню)
├── fonts/
│   └── README.md       — инструкция по подключению шрифта Inter
├── images/
│   ├── favicon.svg     — иконка сайта
│   ├── logo.svg        — логотип FlowAds
│   └── og-image.svg    — превью для соцсетей (OG Image)
└── README.md
```

## Секции сайта

1. **Header** — логотип, навигация, телефон, кнопка CTA, мобильное меню
2. **Hero** — заголовок, статистика, CTA-кнопки, телефон-мокап с картой Яндекс, фоновая карта района
3. **Почему FlowAds** — 6 карточек с честными преимуществами
4. **Услуги** — 4 карточки (SEO, отзывы, ТОП, ведение)
5. **Кейсы** — 3 кейса с мини-графиками роста
6. **Тарифы** — START / BUSINESS / PREMIUM + блок «Индивидуальный план»
7. **Форма заявки** — имя + телефон + согласие, с успех-состоянием
8. **Отзывы** — 3 текстовых + видео-отзыв
9. **Footer** — навигация, контакты, соцсети, копирайт

## Технологии

- Чистый HTML5 + CSS3 + Vanilla JS (без фреймворков)
- CSS Custom Properties (переменные для цветов — легко менять тему)
- Адаптив: Desktop 1280px / Laptop 1100px / Tablet 860px / Mobile 520px
- Шрифт Inter подключён через Google Fonts CDN

## Цветовая палитра (CSS переменные)

```css
--bg:       #0D0D0F   /* фон страницы */
--surface:  #17171C   /* поверхность карточек */
--surface2: #1E1E24   /* альтернативная поверхность */
--purple:   #8B5CF6   /* основной акцент */
--blue:     #3B82F6   /* второй акцент */
--green:    #22C55E   /* позитивные метрики */

```

## Замена контента

Все тексты находятся в `index.html` — просто найдите нужный блок и замените.

Для замены телефона — ищите `+7 (999) 123-45-67` (встречается в header и footer).
Для замены email — ищите `info@flowads.ru`.

## Подключение формы

Форма в текущем виде показывает success-состояние без отправки на сервер.
Для реальной отправки в `js/main.js` в функцию `submitForm()` добавьте fetch:

```js
await fetch('https://ваш-сервер/api/lead', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ name, phone })
});
```

## Запуск

Откройте `index.html` в браузере — и всё готово.
Для продакшна рекомендуется разместить на любом хостинге (Nginx, Apache, Vercel, и т.д.).
